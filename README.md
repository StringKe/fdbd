## 特性

-   [x] 支持 DBML `v1.0`
-   [x] 预览 SQL `v1.0`
-   [x] 设计可视化 `v1.0`
-   [ ] SQL 导入设计 `v1.0`
-   [ ] 团队管理以及项目管理 `v1.0`
-   [ ] 版本管理，差异对比 `v1.0`
-   [ ] DBML 语法检查和高亮 `v1.0`
-   [ ] 自动化生成测试数据 `v2.0`
-   [ ] 多项目依赖分析 `v2.0`
-   [ ] 多语言模型生成 (差异化生成，查询客户端) `v2.0`
-   [ ] IDE 插件支持 (VSCode, Jetbrains) `v2.5`
