## 特性

-   [x] 支持 DBML
-   [x] 预览 SQL
-   [x] 设计可视化
-   [ ] 版本管理，差异对比
-   [ ] 自动化生成测试数据
-   [ ] 多语言模型生成 (差异化生成，查询客户端)
-   [ ] DBML 语法检查和高亮
